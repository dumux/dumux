localAssembler¶

The localAssembler is used to evaluate single entries of the residual vector r and Jacobian Matrix J. The evaluation is done using a localResidual.

Key functionalities¶

  1. assembleJacobianAndResidual()

    1. Calls evalFluxAndSourceResidual() function of the localResidual

    2. Calls evalStorageResidual() function of the localResidual
