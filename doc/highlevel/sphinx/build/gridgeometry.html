gridgeometry¶

A grid geometry is an abstraction of a finite-volume discretization and differs from discretization to discretization. Given a gridview, the grid geometry constructs all geometrical and topological data necessary to evaluate the discrete equation resulting from a given finite volume scheme. All grid geometries inherit from the basic grid geometry.
