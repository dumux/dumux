## localAssembler

|The localAssembler is used to evaluate single entries of the residual vector r and Jacobian Matrix J. The evaluation is done using a {ref}`localResidual`.|
| :--- |

### Key functionalities

1. assembleJacobianAndResidual()
   1. Calls evalFluxAndSourceResidual() function of the {ref}`localResidual`
   2. Calls evalStorageResidual() function of the {ref}`localResidual`
