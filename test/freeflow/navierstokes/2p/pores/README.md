# Two-Phase Flow with Evaporation in Porous Media with Pillars

This test case simulates two-phase flow (water and air) in a channel containing cylindrical pillars with the following features:

## Geometry

- **Channel dimensions**: 1.0m × 0.1m
- **Pillar box**: 0.5m × 0.4m below channel
- **Pillar configuration**: 6×4 hexagonally-packed pillars (alternating rows shifted by half-spacing)
- **Pillar radius**: 0.025m
- **Spacing**: 1 diameter + 1 radius gap between pillar surfaces
- **Mesh refinement**: Fine mesh (lc_pillar = 0.001) around pillars, coarser (lc_box = 0.02) elsewhere

## Physics

### Two-Phase Flow (Stokes equations)
- Incompressible flow of water and air
- Surface tension at water-air interface
- Gravity acting downward

### Initial Conditions
- Water saturation: 0.8 (water fills pores up to pillar level)
- Air saturation: 0.2 (above)

### Boundary Conditions
- **Inlet/Outlet**: No-flow (Neumann)
- **Top surface**: Evaporative flux (depends on surface temperature)
- **Pillar surfaces**: No-slip (Dirichlet velocity = 0)

### Key Features

1. **Evaporation**: Water evaporates from the free surface at rate ~1.0e-5 kg/(m²·s)
2. **Hele-Shaw Drag**: Drag forces around pillar surfaces reduce fluid velocity in narrow gaps
   - Standard Hele-Shaw drag: f = 12η*u / gap²
   - Applied as body force near pillar surfaces

## Running the Simulation

```bash
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
make
./pores main.cc
```

## Output

The simulation produces VTK files containing:
- Velocity field
- Phase saturation (water volume fraction)
- Pressure
- Evaporative flux at free surface
- Drag-induced forces
- Process rank (for parallel runs)

## Physical Phenomena Expected

1. **Capillary imbibition**: Water infiltrates pillar-packed region due to capillary forces
2. **Evaporative drying**: Water gradually evaporates from the free surface
3. **Preferential flow**: Water flows predominantly through larger pores (between pillars)
4. **Interface dynamics**: Meniscus formation around pillars due to surface tension
5. **Hele-Shaw confinement**: Narrow gaps between pillars show reduced flow velocities
